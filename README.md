Quiz Application

How to Run:

1. clone the repository
2. npm init
3. npm start
4. Register
5. login
6. Enjoy!

Features:

1. Register
2. Login
3. Create Quiz
4. Mutiple Question with 4 options with 4 Types of Question
   i) Radio
   ii) Checkbox
   iii) Dropdown
   iv) File Upload
5. Question & Options CRUD operations.
6. Start and End Time for Exam
7. Always Open for exam
8. Update Questions
